Algoritmo de verificación de cédula de indentidad uruguaya

Para calcular el dígito verificador de la cédula uruguaya se toman los primeros siete números y se multiplican cada uno (respetando el lugar, es decir, primero x primero, segundo x el segundo, etc.) por el código 2987634. Luego se toma el resultado y se le busca el número mayor que termine en 0 y se le resta. Ej: 150 - 148 = 2. El número 2 es el dígito verificador para la cédula 1.234.567
